## 2. 쿠폰 유효기간 과거 날짜 허용

쿠폰 등록/수정 화면의 날짜 선택기에서 과거 날짜 선택이 막혀 있던 문제를 수정했다.

- 기존: 날짜 선택 최소값이 오늘 날짜라 지난 날짜 선택 불가
- 변경: 날짜 선택 최소값을 `2000-01-01`로 확장
- 결과: 만료된 쿠폰도 직접 등록 가능
- OCR/이미지 스캔으로 과거 유효기간이 인식되는 경우에도 그대로 선택값에 반영 가능

관련 파일:

- `app/lib/features/coupons/presentation/screens/coupon_create_screen.dart`

## 4. iOS 알림 클릭 상세 이동 보강

iPhone에서 알림 메시지를 클릭했을 때 해당 쿠폰 상세 화면으로 이동하지 않는 문제를 확인하고, 알림 payload 처리 타이밍을 보강했다.

원인 후보:

- iOS에서 알림 클릭 payload가 앱 초기화 또는 스플래시 화면 완료 전에 들어오는 경우가 있음
- 이 시점에 `navigatorKey.currentState`가 아직 준비되지 않으면 상세 화면 이동 요청이 유실될 수 있음

수정 내용:

- 네비게이터가 준비되지 않았을 때 알림 payload를 pending 상태로 보관
- 앱 첫 프레임 이후 또는 앱이 resume될 때 pending payload를 다시 처리
- 알림 처리 중 중복 탭이 들어와도 순차 처리되도록 `_handlingTap` 플래그 추가
- 예외가 발생해도 처리 플래그가 풀리도록 `finally` 처리

관련 파일:

- `app/lib/services/notification_service.dart`
- `app/lib/app/app.dart`

## 5. 설정 화면 가상 멤버십 추가 버튼

스토어 스크린샷 및 테스트 편의를 위해 설정 화면에 가상 멤버십 생성 버튼을 추가했다.

버튼명:

- `가상 멤버십 추가`

생성 항목:

| 멤버십 | 카드 번호 |
|---|---|
| OK캐시백 | 9100123456789 |
| 해피포인트 | 9200123456789 |
| L.POINT | 9300123456789 |
| CJ ONE | 9400123456789 |

구현 방식:

- 고정 ID 기반으로 저장
- 버튼을 여러 번 눌러도 중복 생성하지 않고 같은 항목을 갱신
- 저장 후 `가상 멤버십을 추가했어요.` 스낵바 표시

관련 파일:

- `app/lib/repositories/membership_repository.dart`
- `app/lib/features/settings/presentation/screens/settings_screen.dart`
- `app/lib/core/resources/app_strings.dart`

## 6. 설정 화면 캡처용 여백 조정

스토어 이미지 캡처 시 알림 설정 섹션과 테스트 버튼 섹션이 너무 붙어 보이지 않도록 여백을 크게 늘렸다.

- 기존 여백: `20`
- 변경 여백: `500`

관련 파일:

- `app/lib/features/settings/presentation/screens/settings_screen.dart`
