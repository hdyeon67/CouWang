## 1. 스토어 제출 문서

`docs/store/` 폴더는 스토어 콘솔 입력과 그래픽 제작 요청에 바로 쓰는 산출물 폴더로 둔다.

| 파일 | 목적 | 주요 내용 |
|---|---|---|
| `docs/store/01_google_play_listing.md` | 구글 플레이 등록정보 | 앱 이름, 짧은 설명, 전체 설명, 키워드, 카테고리, 출시 국가, 릴리스 노트 |
| `docs/store/02_appstore_listing.md` | 애플 앱스토어 등록정보 | 앱 이름, 부제목, 홍보 문구, 설명, 키워드, 개인정보 답변, 심사 메모, 새로운 기능 |
| `docs/store/03_store_image_plan.md` | 이미지/그래픽 제작 기획 | 앱 아이콘, 피처드 그래픽, 스크린샷 8장, 앱 미리보기 영상, 납품 규격 |

### iOS 알림 클릭 라우팅 보강

| 항목 | 내용 |
|---|---|
| 문제 | iPhone에서 알림 메시지를 클릭해도 해당 쿠폰 상세로 이동하지 않는 경우 발생 |
| 원인 추정 | 알림 payload가 앱 초기화 또는 스플래시 완료 전 들어오면 네비게이터가 준비되지 않아 이동 요청이 유실될 수 있음 |
| 변경 | payload를 pending 상태로 보관하고 첫 프레임 이후 또는 resume 시 다시 처리 |
| 추가 안전장치 | 중복 탭 순차 처리용 `_handlingTap`, 예외 발생 시 상태 복구용 `finally` 적용 |
| 관련 파일 | `app/lib/services/notification_service.dart`, `app/lib/app/app.dart` |

## 3. 설정 화면 QA/캡처 도구

### 가상 멤버십 추가 버튼

설정 화면에 `가상 멤버십 추가` 버튼을 추가했다. 스토어 스크린샷 촬영과 멤버십 화면 확인을 빠르게 하기 위한 개발/QA 도구다.

| 멤버십 | 카드 번호 | 저장 ID |
|---|---|---|
| OK캐시백 | 9100123456789 | `virtual_membership_ok_cashbag` |
| 해피포인트 | 9200123456789 | `virtual_membership_happy_point` |
| L.POINT | 9300123456789 | `virtual_membership_l_point` |
| CJ ONE | 9400123456789 | `virtual_membership_cj_one` |

운영 기준:

- 고정 ID로 저장해 버튼을 여러 번 눌러도 중복 생성하지 않는다.
- 기존 항목이 있으면 같은 ID의 데이터를 갱신한다.
- 저장 후 `가상 멤버십을 추가했어요.` 스낵바를 표시한다.

관련 파일:

- `app/lib/repositories/membership_repository.dart`
- `app/lib/features/settings/presentation/screens/settings_screen.dart`
- `app/lib/core/resources/app_strings.dart`

### 설정 화면 캡처 여백

스토어 스크린샷 촬영 시 알림 설정 카드와 테스트 버튼 묶음이 한 화면에 과밀하게 보이지 않도록 큰 여백을 적용했다.

| 항목 | 값 |
|---|---|
| 기존 여백 | `20` |
| 변경 여백 | `500` |
| 관련 파일 | `app/lib/features/settings/presentation/screens/settings_screen.dart` |
