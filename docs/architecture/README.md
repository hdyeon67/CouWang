# 아키텍처 문서

쿠왕 앱의 로컬 데이터 구조와 저장 기준을 관리합니다.

| 파일 | 역할 |
|---|---|
| `local_data_erd.md` | 쿠폰, 멤버십, 이미지, 알림 설정/로그의 관계 정의 |
| `local_dataset_spec.md` | 로컬 데이터셋별 필드와 의미 정의 |

## 관리 기준

- 앱 구현과 DB 스키마가 달라지면 이 폴더의 문서를 함께 갱신합니다.
- 사용자 데이터는 로컬 저장을 기본 원칙으로 둡니다.
- 서버, Firebase, 광고 SDK 등 외부 처리와 관련된 개인정보 판단은 `legal/` 문서와 함께 맞춥니다.
