# AI 인계 문서

다른 AI가 쿠왕 프로젝트를 이어받을 때 필요한 요약, 읽기 순서, 복붙용 프롬프트를 관리합니다.

| 파일 | 역할 |
|---|---|
| `01_project_handoff.md` | 프로젝트 전체 맥락, 현재 상태, 핵심 규칙, 우선 확인 파일 |
| `02_ai_handoff_prompt.md` | 다른 AI에 바로 붙여넣을 인계 프롬프트 |

## 사용 순서

1. `01_project_handoff.md`를 읽어 전체 구조와 현재 상태를 파악합니다.
2. `02_ai_handoff_prompt.md`를 새 AI 세션에 붙여넣습니다.
3. 새 AI가 지정된 문서를 읽고 현재 워크트리 상태를 먼저 요약하게 합니다.
