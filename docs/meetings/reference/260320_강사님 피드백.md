## **쿠왕(Couwang)**

**어플 기능**

1. 기프티콘 \+ 오프라인 쿠폰 관리 ← 쿠폰 까먹지 않도록 한다  
2. 멤버십 통합 관리 ← 멤버십을 까먹지 않도록 한다  
3. 사용내역 알림 문구 ← 추가적으로 소비자에게 절약하고 있다는 감정을 준다   
   1.  ex) “이번 달에 쿠폰으로 ㅇㅇㅇ원 사용했어요\!”  
   2. ex) “31일까지 사용 안하면 돈이 땅바닥에 버려져요\!”

**문제**

1. 오프라인 및 모바일 쿠폰 만료일을 놓쳐서 사용하지 못한 적이 있다  
2. 여러 종류의 멤버십을 일일이 찾기 귀찮아서 사용하지 않은 경험이 있다  
3. 멤버십을 찾고 로딩하는 시간이 걸려 계산대에서 딜레이된  경험이 있다

**핵심 가치**

1. 쿠폰을 까먹지 않고 사용한다

2. 멤버십을 까먹지 않고 적립 또는 할인을 받는다

**정체성** 

- 흩어진 쿠폰과 멤버십을 한곳에 모아, 나도 모르게 버려지던 돈을 리마인드 해주는 '내 스마트폰 속  알리미'입니다

**차별성** 

- 기존 앱들이 단순한 '쿠폰 보관함'이라면, 이 어플은 '안 쓰면 버려지는 돈'과 '이번 달 아낀 돈'을 직접적인 금액으로 보여주는  알리미 역할을 합니다.  
- UI 디자인

**타겟**

- 여러 개의 어플 사용이 번거로운  바쁜 3040 직장인

**사용자 니즈**

- 디바이스에 저장되어있는 쿠폰 및 기프티콘의 관리를 쉽게 하고 싶다.   
- 쿠폰 만료일을 놓치고 싶지 않다.   
- 친구/가족들과 쿠폰을 손쉽게 공유하고 싶다.  
- 기존 어플들의 디자인이 예뻤으면 좋겠다.  
- 사용이 편리했으면 좋겠다.(기능이 과하게 많거나 작동하지 않는 기능이 많았다.)

**페인포인트**

- 앨범, 카톡, 문자 등에 흩어진 쿠폰을 일일이 찾기 번거로움  
- 유효기간 임박 알림이 부족해 금전적 손실이 반복적으로 발생함  
- 이미지 저장 후 재전송하는 과정이 복잡하고 사용 여부 확인이 안 됨  
- 기존 앱들의 복잡한 UI와 과도한 광고로 인한 피로도와 이탈 발생

**화면**

- 어플 접속 화면   
- 홈 화면  
  - 쿠폰 리스트 화면  
  - 쿠폰 추가 버튼  
  - 쿠폰 편집 버튼  
  - 리포트 배너 뷰  
    - 쿠폰 사용 금액(월별)  
    - 사용 쿠폰 개수(월별)  
    - 피눈물 나는 버려진 쿠폰 개수(월별)  
- 쿠폰 등록 화면  
  - 수동 입력  
    - 이미지  
    - 카테고리(드롭다운)  
    - 쿠폰번호  
    - 쿠폰명  
    - 사용처  
    - 유효기간  
    - 메모  
  - 자동 입력  
- 쿠폰 상세 화면  
  - 사용완료체크버튼  
  - 수정버튼   
  - 멤버십으로 이동 버튼  
  - 삭제버튼  
- 멤버십 리스트 화면   
  - 멤버십 추가 버튼  
  - 멤버십 편집 버튼  
- 멤버십 등록 화면  
  - 수동입력  
    - 이미지 (이미지 업로드)  
    - 멤버십명  
    - 메모  
- 멤버십 상세 화면  
  - 멤버십 이미지 보여주기  
- 알림 리스트 화면  
  - 리스트 항목 클릭 시 상세 화면으로 이동  
  - 알림 설정 이동 버튼   
- 알림 설정 화면  
  - 쿠폰 만료 알림 on/off,   
  - 당일/1일/3일/7일/30일 전 각각 알림 on/off  
- 팝업(다이얼로그)  
  - 등록하시겠습니까?  
  - 사용완료처리하시겠습니까?  
  - 수정완료하시겠습니까?  
  - 작성중인 내용이 있습니다. 나가시겠습니까?  
  - 쿠폰(멤버십)을 삭제하시겠습니까?  
- 알림 메시지  
  - 클릭 시 상세 화면으로 이동

**멤버십 상세 화면에서 행동 이벤트**

- 예) 올리브영 클릭 \> 올리브영 멤버십 상세 화면 열림(화면 확인) \> 올리브영 멤버십 상세 화면 닫힘  
- 1\. 멤버십 항목 클릭    
- 2\. 멤버십 상세화면 열림  
- 3\. 멤버십 상세화면 닫힘
