# 스토어 제출 문서

Google Play와 App Store 제출에 필요한 등록정보, 이미지 기획, 출시 문구를 관리합니다.

| 파일 | 역할 |
|---|---|
| `01_google_play_listing.md` | Google Play 기본 스토어 등록정보 |
| `02_appstore_listing.md` | Apple App Store 기본 등록정보 |
| `03_store_image_plan.md` | 스토어 스크린샷, 피처드 그래픽, 앱 아이콘 기획 |
| `04_release_notes.md` | Google Play / App Store 출시노트와 빌드 기록 |

## 관리 기준

- 실제 앱 기능과 다르게 보일 수 있는 표현은 사용하지 않습니다.
- 개인정보처리방침 URL, 지원 URL, 연락처는 `legal/` 문서와 같은 값으로 유지합니다.
- AdMob/Firebase 등 SDK 추가 시 광고 포함 여부와 개인정보 설명을 함께 갱신합니다.
