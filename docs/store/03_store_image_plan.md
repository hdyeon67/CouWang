## 1. 앱 아이콘

### 디자인 방향

- 쿠왕 마스코트(검은 강아지) 중심
- 배경: 흰색 또는 `#64CAFA` 하늘색
- 형태: 라운드 사각형 (iOS/Android 공통 원본 1024x1024px)
- 심플하고 인식하기 쉽게 구성
- 작은 사이즈에서도 강아지 얼굴, 귀, 눈, 목줄 포인트가 뭉개지지 않도록 선명한 실루엣 유지

### 제출 사이즈 목록

| 플랫폼 | 사이즈 | 용도 |
|---|---:|---|
| Android | 512x512px | 구글 플레이 고해상도 아이콘 |
| iOS | 1024x1024px | 앱스토어 아이콘 |
| Android | 48~192px | 앱 내부 런처 아이콘 세트 |

## 2. 피처드 그래픽 (Feature Graphic)

구글 플레이 전용 그래픽입니다. 앱스토어에는 해당 항목이 없습니다.

### 사이즈

- 1024 x 500px (필수)
- 포맷: PNG 또는 JPG
- 최대 용량: 1MB

### 디자인 기획안

#### 안 A: 캐릭터 중심형

- 배경: `#64CAFA` 하늘색 그라디언트
- 중앙: 쿠왕 마스코트 크게 배치
- 우측: 앱 스크린샷 1장(홈 화면)을 살짝 비스듬히 배치
- 상단: `쿠왕` 로고 텍스트
- 하단: 슬로건 `쿠폰이 사라지기 전에, 쿠왕이 먼저 알려드려요`
- 포인트: 알림 뱃지 `D-DAY` 플로팅 요소 추가
- 목적: 앱의 귀여운 첫인상과 만료 알림 정체성을 가장 빠르게 전달

#### 안 B: 절약 강조형

- 배경: 흰색
- 좌측: 쿠왕 마스코트 + 말풍선 `42,500원 아꼈다 멍!`
- 우측: 쿠폰 카드 목업 3장 겹쳐서 배치
- 포인트: 절약 금액 숫자를 `#64CAFA` 컬러로 크게 강조
- 목적: 쿠폰을 잊지 않고 쓰면 실제 절약으로 이어진다는 메시지 강조

#### 안 C: 알림 강조형

- 배경: 딥 블루 `#1A2A3A`에서 하늘색으로 이어지는 그라디언트
- 중앙: 스마트폰 잠금화면 + 쿠왕 알림 메시지 노출 목업
- 상단: `놓친 쿠폰, 이제 없어요` 메인 카피
- 우측 하단: 쿠왕 마스코트 작게 배치
- 포인트: 알림 카드에 `오늘 만료되는 쿠폰이 있어요` 문구 사용
- 목적: 앱의 핵심 기능인 만료 전 알림을 직관적으로 전달

## 3. 스크린샷 기획

### 사이즈 기준

| 플랫폼 | 권장 사이즈 | 방향 | 최대 장수 |
|---|---:|---|---:|
| 구글 플레이 | 1080x1920px | 세로 | 8장 |
| 앱스토어 6.5" | 1242x2688px | 세로 | 10장 |
| 앱스토어 5.5" | 1242x2208px | 세로 | 10장 |

### 스크린샷 구성

총 8장 구성으로, 구글 플레이와 앱스토어에 공통 활용합니다. 앱스토어 제출 시 6.5형과 5.5형 규격에 맞게 동일 디자인을 리사이징합니다.

#### 1장. 메인 인트로 컷

- 배경: `#64CAFA`
- 쿠왕 마스코트 중앙 배치
- 대형 카피: `쿠폰, 이제 놓치지 마세요`
- 서브 카피: `만료 전에 쿠왕이 먼저 알려드려요`
- 실제 앱 화면 없음, 순수 그래픽 컷
- 역할: 스토어 첫 화면에서 앱의 문제 해결 가치를 즉시 전달

#### 2장. 홈 화면 (쿠폰 리스트)

- 화면: 실제 홈 화면 스크린샷
- 상단 캡션: `내 쿠폰을 한눈에`
- 하단 설명: `만료순 정렬 · 검색 · D-DAY 뱃지로 한번에 확인`
- 포인트 표시: D-DAY 뱃지에 말풍선 강조
- 역할: 쿠폰 리스트, 만료일, 검색 흐름을 보여주는 핵심 기능 컷

#### 3장. 쿠폰 상세 + 바코드

- 화면: 실제 쿠폰 상세 화면 스크린샷
- 상단 캡션: `바코드도 바로 꺼내요`
- 하단 설명: `계산대 앞에서 당황하지 않도록,\n탭 한 번으로 바코드 확대`
- 포인트 표시: 바코드/QR 미리보기 영역 확대 강조
- 역할: 실제 사용 순간의 편리함 전달

#### 4장. 만료 알림 화면

- 화면: 잠금화면 + 쿠왕 알림 메시지 목업
- 상단 캡션: `만료 전에 먼저 알려드려요`
- 하단 설명: `D-30부터 D-DAY까지\n단계별로 놓치지 않게`
- 알림 예시: `쿠왕 알림 · 오늘 만료되는 쿠폰이 있어요`
- 역할: 만료 전 알림 기능을 가장 직관적으로 시각화

#### 5장. 알림 리스트 화면

- 화면: 실제 NotificationListScreen 스크린샷
- 상단 캡션: `최근 알림을 한 곳에서`
- 하단 설명: `어떤 쿠폰이 곧 만료되는지\n알림 리스트에서 바로 확인`
- 포인트 표시: 읽지 않은 알림 카드와 D-Day 뱃지 강조
- 역할: 알림 기록과 상세 이동 흐름 안내

#### 6장. 멤버십 관리 화면

- 화면: 실제 MembershipListScreen 스크린샷
- 상단 캡션: `멤버십도 함께 관리해요`
- 하단 설명: `CJ ONE, 스타벅스, SKT 등\n자주 쓰는 멤버십 바코드도 한곳에`
- 포인트 표시: 멤버십 카드 리스트 또는 바코드 진입 요소 강조
- 역할: 쿠폰 앱을 넘어 생활 바코드 관리 앱으로 확장되는 가치 전달

#### 7장. 쿠폰 등록 화면

- 화면: 실제 CouponCreateScreen 스크린샷
- 상단 캡션: `등록도 간편하게`
- 하단 설명: `이미지 업로드 후\nOCR로 정보 자동 추출`
- 포인트 표시: `이미지에서 정보 추출하기` 버튼 강조
- 역할: 수동 입력 부담을 줄이는 자동 입력 보조 기능 전달

#### 8장. 절약 금액 강조 컷

- 화면: 홈 화면의 절약 말풍선 카드 클로즈업 또는 그래픽 컷
- 배경: `#64CAFA`
- 대형 카피: `이번 달 42,500원 아꼈다 멍! 🐾`
- 서브 카피: `쿠왕과 함께라면 쿠폰이 돈이 돼요`
- 포인트 표시: 금액 숫자는 크게, 쿠왕 마스코트는 말풍선 옆 배치
- 역할: 쿠폰 관리의 감정적 보상과 절약 체감 강조

## 4. 스크린샷 공통 디자인 가이드

### 프레임 스타일

- 스마트폰 목업 프레임: 사용하지 않음 (풀스크린 방식)
- 상단 캡션 영역:
  - 높이: 약 180px
  - 배경: `#64CAFA` 또는 흰색 (장수별 교차)
  - 폰트: Bold
  - 텍스트 컬러: 흰색 또는 `#1A1A1A`
  - 정렬: 중앙
- 하단 설명 영역:
  - 높이: 약 140px
  - 폰트: Regular
  - 텍스트 컬러: `#555555`
  - 정렬: 중앙
- 실제 앱 화면 영역: 중간 메인 공간 차지
- 모든 문구는 모바일 썸네일에서도 읽히도록 짧고 굵게 구성

### 컬러 팔레트

| 역할 | 색상 |
|---|---|
| 강조색 | `#64CAFA` |
| 보조색 | `#FFFFFF` |
| 배경 | `#F5F5F7` |
| 메인 텍스트 | `#1A1A1A` |
| 서브 텍스트 | `#555555` |
| 포인트 | `#FFD966` |

### 장수별 배경 패턴

| 장수 | 캡션 배경 |
|---:|---|
| 1, 3, 5, 7 | `#64CAFA` |
| 2, 4, 6, 8 | 흰색 |

스토어 썸네일이 가로로 나열될 때 하늘색과 흰색이 번갈아 보여 리듬감이 생기도록 구성합니다.

### iPad 스크린샷 (선택)

- 사이즈: 2048x2732px
- 태블릿 레이아웃 미지원 시 iPhone 스크린샷 기반 제출 가능
- 향후 태블릿 대응 시 iPad 전용 레이아웃으로 별도 기획

### 앱 미리보기 영상 (App Preview, 선택)

- 길이: 15~30초
- 방향: 세로
- 권장 사이즈: 886x1920px
- 포맷: MP4 또는 MOV

| ② 홈 화면 | 3초 | 쿠폰 리스트 스크롤, D-DAY 뱃지 노출 |
