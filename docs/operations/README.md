# 운영 문서

앱 실행, 디바이스 연결, 릴리즈 빌드, 내부테스트 배포에 필요한 운영 절차를 관리합니다.

| 파일 | 역할 |
|---|---|
| `run_and_release_guide.md` | Android/iOS/Web 실행, Firebase 옵션, AAB/IPA 빌드 가이드 |

## 관리 기준

- 실제로 성공한 명령어를 우선 기록합니다.
- 기기 ID, 포트, 빌드 결과처럼 자주 바뀌는 값은 예시와 함께 최신 작업 노트에도 남깁니다.
- 스토어 제출에 영향을 주는 빌드 옵션은 `store/`와 `legal/` 문서의 내용과 맞춥니다.
