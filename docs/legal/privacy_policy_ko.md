# 쿠왕 개인정보처리방침

시행일: 2026년 4월 21일

쿠왕(Couwang, 이하 "앱")은 이용자의 개인정보 보호를 중요하게 생각합니다. 본 개인정보처리방침은 앱이 어떤 정보를 처리하는지, 해당 정보가 어디에 저장되는지, 이용자가 어떤 권리를 가지는지 안내하기 위해 작성되었습니다.

쿠왕은 로그인, 회원가입, 쿠왕 서버 저장 기능을 제공하지 않습니다. 이용자가 앱에 입력하거나 선택한 쿠폰 및 멤버십 정보는 이용자의 기기 안에 로컬로 저장되며, 개발자 서버로 전송되지 않습니다. 다만 앱에는 Google AdMob 배너 광고가 포함될 수 있으며, 광고 제공 및 부정 사용 방지, 광고 성과 측정을 위해 Google Mobile Ads SDK가 일부 기기 및 사용 정보를 처리할 수 있습니다.

## 1. 개인정보처리방침 개요

쿠왕은 쿠폰, 기프티콘, 멤버십 바코드를 기기 내에서 관리하고 만료 전 알림을 받을 수 있도록 돕는 앱입니다.

앱은 개인을 식별할 수 있는 이름, 이메일 주소, 전화번호, 계정 정보, 정확한 위치 정보 등을 직접 수집하지 않습니다. 쿠폰, 기프티콘, 멤버십 정보는 기기 내에만 저장됩니다. 광고 제공을 위해 Google AdMob SDK가 광고 식별자, 기기 정보, 앱 상호작용, 진단 정보 등을 처리할 수 있으며, 자세한 내용은 Google의 광고 및 개인정보 정책을 따릅니다.

### 2.2 기기 내 저장 데이터

이용자가 앱 기능을 사용하기 위해 직접 입력하거나 선택한 아래 정보는 이용자의 기기 안에만 저장됩니다.

| 저장 데이터 | 예시 | 저장 위치 | 외부 전송 |
|---|---|---|---|
| 쿠폰/기프티콘 정보 | 쿠폰명, 브랜드, 유효기간, 바코드/QR 코드값, 메모 | 이용자 기기 내 로컬 저장소 | 없음 |
| 멤버십 정보 | 멤버십명, 브랜드, 카드번호, 메모 | 이용자 기기 내 로컬 저장소 | 없음 |
| 이미지 정보 | 이용자가 선택한 쿠폰/멤버십 이미지 | 이용자 기기 내 앱 저장공간 | 없음 |
| 알림 설정 | 알림 ON/OFF, 알림 주기 | 이용자 기기 내 로컬 저장소 | 없음 |
| 알림 기록 | 만료 알림 제목, 알림 내용, 읽음 여부 | 이용자 기기 내 로컬 저장소 | 없음 |

위 쿠폰/멤버십 데이터는 앱 기능 제공을 위해 기기 내에서만 처리되며, 개발자 서버 또는 제3자 서버로 전송되지 않습니다.

### 2.3 광고 SDK 처리 정보

앱은 무료 서비스 운영을 위해 Google AdMob 배너 광고를 표시할 수 있습니다. Google Mobile Ads SDK는 광고 제공, 광고 성과 측정, 보안 및 부정 사용 방지를 위해 아래와 같은 정보를 자동으로 처리할 수 있습니다.

| 처리될 수 있는 정보 | 예시 | 처리 주체 |
|---|---|---|
| 기기 및 광고 식별자 | 광고 ID, 앱 세트 ID, 기기 관련 식별자 | Google AdMob |
| 네트워크 정보 | IP 주소를 통한 대략적 위치 추정 | Google AdMob |
| 앱 상호작용 | 앱 실행, 광고 노출/탭 등 광고 관련 상호작용 | Google AdMob |
| 진단 정보 | 앱 및 SDK 성능, 오류, 중단 정보 | Google AdMob |

쿠왕 개발자는 쿠폰/멤버십 원본 데이터를 광고 SDK에 제공하지 않습니다.

광고 표시를 위해 Google AdMob SDK가 기기 및 광고 관련 정보를 처리할 수 있습니다. 이 처리는 광고 제공, 광고 성과 측정, 보안 및 부정 사용 방지 목적에 한정됩니다.

## 6. 개인정보 처리 위탁 여부

쿠왕은 쿠폰/멤버십 데이터 처리를 외부 업체에 위탁하지 않습니다.

다만 앱 내 배너 광고 제공을 위해 Google AdMob을 사용합니다. AdMob SDK가 처리하는 광고 관련 정보는 Google의 개인정보처리방침 및 Google Mobile Ads SDK 정책에 따릅니다.

## 7. 이용자 권리

이용자는 앱 내에서 직접 저장한 쿠폰, 멤버십, 알림 기록을 확인, 수정, 삭제할 수 있습니다.

이용자는 기기 설정에서 앱 권한을 언제든지 변경할 수 있습니다.

| 권리 | 행사 방법 |
|---|---|
| 데이터 확인 | 앱 내 쿠폰/멤버십/알림 화면에서 확인 |
| 데이터 수정 | 각 상세 화면의 수정 기능 이용 |
| 데이터 삭제 | 각 상세 화면 또는 알림 화면의 삭제 기능 이용 |
| 권한 철회 | 기기 설정에서 사진/카메라/알림 권한 변경 |
| 전체 삭제 | 앱 삭제 시 기기 내 앱 데이터 삭제 |

앱이 서버에 데이터를 저장하지 않으므로, 개발자는 이용자의 기기 내 데이터를 원격으로 조회, 수정, 삭제할 수 없습니다.
