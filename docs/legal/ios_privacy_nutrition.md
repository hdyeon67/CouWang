# 앱 개인정보 라벨 입력 가이드

이 문서는 App Store Connect의 **앱 개인정보 처리 방침(App Privacy / Privacy Nutrition Label)** 입력 시 참고하기 위한 가이드입니다.

쿠왕은 로그인, 서버 저장, 광고, 사용자 추적이 없으며 쿠폰/멤버십/이미지 데이터는 기기 안에 로컬로 저장됩니다.

## 1. 입력 전 기준 정리

| 기준 | 쿠왕 현재 정책 |
|---|---|
| 계정/로그인 | 없음 |
| 서버 저장 | 없음 |
| 외부 전송 | 없음 |
| 광고 | 없음 |
| 사용자 추적 | 없음 |
| 쿠폰/멤버십 데이터 | 기기 내 로컬 저장 |
| 이미지 데이터 | 사용자가 선택한 이미지를 기기 내 앱 저장공간에 저장 |
| 알림 | 서버 푸시가 아닌 로컬 알림 |

## 2. 데이터 수집 여부

### 선택값

**데이터를 수집하지 않음**

### 이유

쿠왕은 사용자 데이터를 개발자 서버, 제3자 서버, 광고 네트워크, 분석 서비스로 전송하지 않습니다. 사용자가 입력하거나 선택한 쿠폰, 멤버십, 이미지 정보는 앱 기능 제공을 위해 기기 내에만 저장됩니다.

## 3. 중요 입력 주의사항

App Store Connect에서 처음에 **데이터를 수집하지 않음**을 선택하면, 일반적으로 세부 데이터 유형을 별도로 선택하지 않습니다.

다만 화면 흐름 또는 심사 대응 과정에서 사진/동영상 항목을 별도로 설명해야 하는 경우를 대비해 아래 보조 기준을 함께 정리합니다.

권장 입력은 다음과 같습니다.

| 상황 | 권장 선택 |
|---|---|
| 이미지가 기기 내에만 저장되고 외부 전송이 전혀 없는 현재 구조 | 데이터를 수집하지 않음 |
| 향후 이미지가 서버, 클라우드, 분석 도구, OCR 서버 등 외부로 전송되는 구조로 바뀌는 경우 | 사용자 콘텐츠 > 사진 또는 동영상 수집함 |

현재 쿠왕 v1.0 기준으로는 **데이터를 수집하지 않음**을 선택하는 것이 앱 구조와 가장 일치합니다.

## 4. 데이터 유형별 입력 가이드

아래 표는 App Store Connect 화면 순서에 맞춘 확인용 체크리스트입니다.

| 데이터 유형 | 선택값 | 이유 |
|---|---|---|
| 연락처 정보 | 수집 안 함 | 이름, 이메일, 전화번호, 주소를 수집하지 않음 |
| 건강 및 피트니스 | 수집 안 함 | 건강/운동 관련 기능 없음 |
| 재무 정보 | 수집 안 함 | 결제, 신용카드, 금융 정보 기능 없음 |
| 위치 | 수집 안 함 | 위치 기반 기능 없음 |
| 민감한 정보 | 수집 안 함 | 민감정보 입력 또는 처리 기능 없음 |
| 연락처 | 수집 안 함 | 기기 연락처 접근 없음 |
| 사용자 콘텐츠 | 수집 안 함 | 현재 구조에서는 사진/쿠폰 이미지를 외부로 전송하지 않고 기기 내에만 저장 |
| 식별자 | 수집 안 함 | 사용자 ID, 기기 ID, 광고 ID를 수집하지 않음 |
| 구매 | 수집 안 함 | 인앱 구매 또는 결제 기능 없음 |
| 사용 데이터 | 수집 안 함 | 외부 분석 또는 사용 로그 전송 없음 |
| 진단 | 수집 안 함 | 개발자 서버로 충돌/성능 진단 데이터 전송 없음 |
| 기타 데이터 | 수집 안 함 | 위 항목 외 수집 데이터 없음 |

## 5. 사용자 콘텐츠 항목 보조 기준

요청서 기준의 "사진 또는 동영상" 항목을 App Store Connect에서 별도로 입력해야 하는 상황이 생기면 아래처럼 판단합니다.

### 현재 v1.0 권장 선택

| 항목 | 선택값 | 이유 |
|---|---|---|
| 사용자 콘텐츠 > 사진 또는 동영상 | 수집 안 함 | 이미지는 사용자의 기기 내 앱 저장공간에만 저장되고 개발자 또는 제3자에게 전송되지 않음 |

### 향후 서버 전송 기능이 생기는 경우의 대체 선택

| 항목 | 선택값 |
|---|---|
| 사용자 콘텐츠 > 사진 또는 동영상 | 수집함 |
| 목적 | 앱 기능 |
| 사용자 식별에 연결 | 아니오 |
| 추적 목적 | 아니오 |

이 대체 선택은 현재 쿠왕 v1.0에는 적용하지 않습니다. 향후 이미지가 외부 서버로 전송되는 구조가 생길 때만 사용합니다.

## 6. 추적 여부

### 선택값

**이 앱은 사용자 또는 기기를 추적하지 않습니다**

### 이유

쿠왕은 광고 식별자, 사용자 ID, 기기 ID를 사용하지 않으며, 사용자의 데이터를 제3자 데이터와 결합하거나 광고 목적으로 추적하지 않습니다.

## 7. 개인정보 처리방침 URL

### 선택값

개인정보처리방침을 배포한 공개 URL 입력

### 입력 전 준비

| 항목 | 상태 |
|---|---|
| 한국어 개인정보처리방침 | `docs/legal/privacy_policy_ko.md` 기준으로 웹 게시 필요 |
| 영어 Privacy Policy | `docs/legal/privacy_policy_en.md` 기준으로 웹 게시 필요 |
| 한국어 HTML 예상 URL | `https://hdyeon67.github.io/CouWang/legal/privacy_policy_ko.html` |
| 영어 HTML 예상 URL | `https://hdyeon67.github.io/CouWang/legal/privacy_policy_en.html` |
| 대표 개인정보처리방침 URL | `https://hdyeon67.github.io/CouWang/legal/privacy_policy_ko.html` |

## 8. 최종 제출 전 체크리스트

| 체크 항목 | 확인 |
|---|---|
| 로그인/회원가입이 없는가 | 예 |
| 서버 또는 Firebase로 사용자 데이터가 전송되지 않는가 | 예 |
| 광고 SDK 또는 추적 SDK가 없는가 | 예 |
| 사진/이미지가 외부 서버로 업로드되지 않는가 | 예 |
| 알림이 로컬 알림으로만 동작하는가 | 예 |
| 개인정보처리방침 URL이 공개 접근 가능한가 | 제출 전 확인 필요 |
| App Store Connect에서 "데이터를 수집하지 않음" 선택이 현재 구현과 일치하는가 | 예 |
