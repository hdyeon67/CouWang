# 법적 문서와 심사 문서

스토어 제출, 개인정보처리방침 URL, App Store 심사 대응에 필요한 문서를 관리합니다.

| 파일 | 역할 |
|---|---|
| `privacy_policy_ko.md` | 한국어 개인정보처리방침 원문 |
| `privacy_policy_en.md` | 영어 Privacy Policy 원문 |
| `privacy_policy_ko.html` | GitHub Pages 게시용 한국어 개인정보처리방침 |
| `privacy_policy_en.html` | GitHub Pages 게시용 영어 Privacy Policy |

| `ios_privacy_nutrition.md` | App Store 개인정보 라벨 입력 가이드 |

## 관리 기준

- Markdown 원문을 기준으로 내용을 관리하고, 공개 제출이 필요한 경우 HTML도 함께 갱신합니다.
- AdMob, Firebase, 외부 SDK가 추가되면 개인정보처리방침과 개인정보 라벨 가이드를 반드시 다시 확인합니다.
- 연락처, 주소, 개발사 정보는 스토어 등록정보와 같은 값으로 유지합니다.
