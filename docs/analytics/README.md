# 분석 문서

쿠왕의 이벤트 이름, KPI 계산식, 분석 해석 기준을 관리합니다.

| 파일 | 역할 |
|---|---|
| `event_spec.md` | 앱/DB/분석에서 공통으로 사용할 이벤트 명세 |
| `kpi_definition.md` | 쿠폰 사용 완료율, 등록 완료율, 알림 오픈율 등 KPI 정의 |

## 관리 기준

- 이벤트 이름은 앱 코드, SQL, 대시보드에서 동일하게 유지합니다.
- 사용자 입력값, 바코드 번호, 이미지 경로처럼 민감하거나 개인적인 값은 이벤트 속성에 넣지 않습니다.
- Firebase Analytics를 켜는 경우에도 이 문서의 이벤트 목적과 개인정보 기준을 먼저 확인합니다.
