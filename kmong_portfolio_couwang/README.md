# 쿠왕 크몽 포트폴리오 산출물

이 폴더는 크몽 포트폴리오 등록에 바로 활용할 수 있도록 정리한 문구와 이미지 템플릿을 포함합니다.

## PNG 파일 대신 SVG를 제공한 이유

현재 세션에서는 크몽 업로드용 PNG를 직접 내보내기보다, 문구 수정과 재활용이 쉬운 SVG 템플릿을 제공하는 편이 더 안정적입니다.

## SVG를 PNG로 변환하는 방법

### 방법 1. Figma / Illustrator / Sketch

1. SVG 파일 열기
2. 텍스트와 색상 최종 점검
3. `Export` -> `PNG`
4. 2x 또는 3x 해상도로 저장

### 방법 2. 브라우저에서 열고 캡처

1. SVG 파일을 브라우저에서 열기
2. 확대율 100% 확인
3. 스크린샷 또는 export 도구 사용

### 방법 3. macOS 미리보기

1. SVG 파일 열기
2. PDF로 출력
3. PNG로 다시 변환

## 권장 업로드 흐름

1. `kmong_input_values.md`에서 포트폴리오 입력값 복사
2. `portfolio_content.md`에서 상세 설명 복사
3. `tech_stack_recommendation.md`에서 직업/기술 정보 선택
4. SVG 템플릿을 PNG로 export 후 업로드
