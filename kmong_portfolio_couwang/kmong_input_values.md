## 포트폴리오 제목

쿠왕(Couwang) | Flutter 기반 쿠폰 관리 앱 기획·개발·Android 출시 프로젝트

## 짧은 소개문

쿠폰과 기프티콘의 만료일을 놓치지 않도록 돕는 Flutter 기반 앱 프로젝트입니다. 서비스 기획부터 화면 설계, Flutter 개발, Google Play 출시, 스토어 문서와 홍보 콘텐츠 기획까지 직접 수행했습니다.

## 프로젝트 개요

쿠왕은 사용자가 받은 쿠폰·기프티콘·멤버십 바코드를 한곳에 모아 관리하고, 유효기간 전에 로컬 알림으로 사용을 돕는 모바일 앱입니다. 로그인 없이 사용할 수 있고, 주요 데이터는 기기 로컬에 저장되도록 설계했습니다.

## 문제 정의

모바일 쿠폰과 기프티콘은 카카오톡, 문자, 캡처 이미지, 브랜드 앱 등 여러 경로로 흩어져 저장되는 경우가 많습니다. 이 때문에 보유 중인 쿠폰을 제때 찾지 못하거나, 유효기간을 놓쳐 사용하지 못하는 문제가 반복됩니다.

## 해결 방향

쿠왕은 쿠폰 정보를 직접 입력하거나 갤러리 이미지에서 읽어 등록할 수 있도록 구성하고, D-day 기준 만료 알림과 바코드/QR 표시 흐름을 결합해 “관리”에서 끝나지 않고 “실사용”까지 이어지는 구조를 목표로 설계했습니다.

## 주요 기능

- 쿠폰 / 기프티콘 등록 및 목록 관리
- 유효기간 기준 D-day 표시
- D-30 / D-7 / D-3 / D-1 / 당일 만료 로컬 알림
- 바코드 / QR 코드 생성 및 상세 화면 표시
- 갤러리 이미지 선택 후 OCR / 바코드 인식 기반 자동 입력 보조
- 멤버십 바코드 저장 및 조회
- 최근 알림 리스트 확인
- 로컬 저장 기반 사용, 로그인 없이 시작 가능

## 사용자 흐름

1. 앱 실행 후 홈 대시보드 진입
2. 쿠폰 등록 화면에서 수동 입력 또는 갤러리 이미지 선택
3. OCR / 바코드 인식으로 정보 자동 보조 입력
4. 저장 후 홈 화면에서 D-day 기준으로 쿠폰 확인
5. 상세 화면에서 바코드 / QR 확인
6. 만료 전 로컬 알림 수신
7. 멤버십 화면에서 멤버십 바코드 함께 관리
