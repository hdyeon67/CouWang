# 상세 이미지 2 기획

이미지명: 상세 이미지 2  
권장 크기: 1200 x 1600px

상단 문구:
- 프로젝트에서 맡은 역할

본문 문구:
- 서비스 기획
- 화면 설계
- Flutter 개발
- Android 공개
- iOS 확장 가능 구조
- 스토어 문서 및 홍보 콘텐츠 기획

하단 문구:
- 아이디어 정리부터 구현, 공개 준비까지 한 흐름으로 수행

디자인 방향:
- 좌측에는 역할 리스트, 우측에는 단계형 프로세스 다이어그램
- 기획 → 설계 → 개발 → 공개 문서 → 출시 대응 순서가 한눈에 보이게 구성

포함하면 좋은 시각 요소:
- 와이어프레임 모티프
- Flutter 로고 느낌의 개발 아이콘
- Google Play 배지
- 지원 URL / 문서 카드 형태 요소
