# 상세 이미지 1 기획

이미지명: 상세 이미지 1  
권장 크기: 1200 x 1600px

상단 문구:
- 어떤 문제를 해결하는 앱인가

본문 문구:
- 문제
  - 쿠폰과 기프티콘이 여러 채널에 흩어져 관리가 어렵다
  - 유효기간을 놓쳐 사용하지 못하는 경우가 많다
  - 계산대 앞에서 바코드와 멤버십을 급하게 찾게 된다
- 해결
  - 쿠폰 / 기프티콘 / 멤버십 바코드를 한곳에서 관리
  - 만료 전 로컬 알림으로 사용 시점 확보
  - 이미지 기반 OCR / 바코드 인식으로 등록 부담 완화
- 핵심 기능
  - 쿠폰 등록
  - D-day 관리
  - 알림
  - 바코드 / QR 표시
  - 멤버십 관리

하단 문구:
- 아이디어를 실제 사용 흐름으로 연결한 모바일 서비스 설계 사례

디자인 방향:
- 3단 카드형 섹션: 문제 / 해결 / 핵심 기능
- 텍스트 위주이되 앱 스크린샷 일부를 함께 사용
- 정보 전달 중심, 과장 없는 포트폴리오 톤

포함하면 좋은 시각 요소:
- 쿠폰 리스트 화면
- 쿠폰 상세 바코드 화면
- 알림 아이콘 또는 일정 캘린더 모티프
