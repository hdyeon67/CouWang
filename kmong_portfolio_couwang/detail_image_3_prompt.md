# 상세 이미지 3 기획

이미지명: 상세 이미지 3  
권장 크기: 1200 x 1600px

상단 문구:
- 외주 서비스로 어떻게 연결되는가

본문 문구:
- 앱/웹 아이디어를 개발 가능한 기획서와 견적서로 정리
- 서비스 소개용 반응형 페이지 제작 가능
- Flutter 기반 Android/iOS MVP 화면 제작 가능
- 출시 전 구조 검토 및 스토어 대응 문서 정리 가능

하단 문구:
- 초기 아이디어부터 실행 가능한 산출물까지 연결해드립니다

디자인 방향:
- 4개 서비스 박스를 명확히 분리
- “기획”, “페이지”, “MVP”, “출시 준비” 흐름으로 정리
- 상담 전환을 염두에 두되 과장 없이 전문적인 톤 유지

포함하면 좋은 시각 요소:
- 문서 아이콘
- 브라우저 프레임
- 모바일 목업
- 체크리스트 / 검토 아이콘
