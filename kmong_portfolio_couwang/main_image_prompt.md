# 메인 이미지 기획

이미지명: 메인 이미지  
권장 크기: 1200 x 900px

상단 문구:
- 실사용까지 고려한 쿠폰 관리 앱 프로젝트

본문 문구:
- 쿠왕(Couwang)
- Flutter 기반 쿠폰·기프티콘 관리 앱

하단 문구:
- 서비스 기획 · Flutter 개발 · Android 공개 · iOS 확장 가능

디자인 방향:
- 브랜드 컬러 `#64CAFA` 중심의 밝고 신뢰감 있는 톤
- 좌측에는 프로젝트 개요, 우측에는 모바일 화면 2~3개 미리보기 배치
- “기획부터 개발, 공개 문서까지 직접 진행” 메시지가 자연스럽게 보이도록 구성

포함하면 좋은 시각 요소:
- 앱 홈 화면 썸네일
- 쿠폰 카드 UI
- 바코드 또는 QR 코드 모티프
- 검은 강아지 마스코트

권장 문구 예시:
- 메인 제목: 쿠왕 | Flutter 기반 쿠폰 관리 앱 프로젝트
- 보조 제목: 기획부터 Android 공개, 지원 문서 구성까지 직접 진행한 서비스 사례
- 하단 키워드: 서비스 기획 / Flutter / Android / iOS 확장 가능
