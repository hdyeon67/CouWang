# Changelog

## 2026-04-05

### 권한 및 시작 흐름
- 스플래시에서 홈으로 진입하기 전에 알림 권한과 사진 접근 권한을 확인하도록 정리했습니다.
- 알림 권한이 비허용 상태일 때 설정 화면의 알림 스위치가 기본적으로 모두 `off`로 보이도록 수정했습니다.
- 설정 화면에서 알림을 다시 켜려는 경우 권한 요청 흐름을 먼저 거치도록 연결했습니다.
- 쿠폰/멤버십 등록 화면에서 이미지 추가 또는 변경 시 사진 권한이 없으면 먼저 권한 요청을 진행하도록 정리했습니다.

### 쿠폰
- CouponDetailScreen을 커스텀 헤더, 이미지/바코드 확대, 사용 완료 처리, 멤버십 바텀시트 구조로 재구성했습니다.
- CouponCreateScreen을 리디자인하고, 링크 입력 시 바코드 대신 QR 코드가 자동으로 표시되도록 개선했습니다.
- 쿠폰 리스트의 정렬 버튼을 팝업 메뉴 대신 클릭 토글형 버튼으로 변경했습니다.
- 좁은 너비 디바이스에서 필터 칩과 정렬 버튼이 overflow 되지 않도록 반응형 레이아웃을 적용했습니다.
- 상세 화면 D-day 뱃지에 `사용 완료`, `기간만료` 상태 표현을 추가했습니다.

### 멤버십
- MembershipDetailScreen을 새 상세 레이아웃으로 구현하고 수정/삭제/쿠폰함 연계 흐름을 정리했습니다.
- MembershipCreateScreen을 이미지 선택 중심 폼으로 재구성하고 수정 프리필 흐름을 연결했습니다.

### 알림 / 설정 / 공통
- NotificationListScreen에 개별 삭제와 전체 삭제 확인 다이얼로그를 추가했습니다.
- 읽은 알림과 만료/사용 완료 알림도 쿠폰 상세 화면으로 이동하도록 정리했습니다.
- 주요 사용자 노출 문자열을 `AppStrings` 기반으로 공통 리소스 관리하도록 정리했습니다.

### 자산 및 플랫폼
- Android/iOS 앱 아이콘과 런치 관련 자산을 `app_icon.png` 기준으로 재정리했습니다.
- iOS `AppIcon.appiconset`을 실제 앱 아이콘 세트 기준으로 다시 생성했습니다.
- 권한 처리 추가에 맞춰 Flutter 의존성과 플랫폼 설정을 업데이트했습니다.

## 2026-04-03

### 앱 구조 및 공통 UI
- 홈, 멤버십, 설정 화면에서 사용하는 floating 하단 탭 구조를 재정리했습니다.
- FAB와 하단 탭이 서로 겹치거나 터치 영역이 꼬이던 문제를 줄이기 위해 공통 scaffold 레이어를 조정했습니다.
- 스플래시 화면과 네이티브 런치 배경 톤을 맞추고, 스플래시 마스코트 자산을 정리했습니다.

### 홈 / 쿠폰
- HomeDashboardScreen 디자인을 쿠폰함 중심 구조로 전면 리디자인했습니다.
- 알림 이동, 정렬, 검색, 상세 이동, 쿠폰 추가 흐름을 다시 연결했습니다.
- 쿠폰 등록 화면을 새 입력 폼 구조로 리디자인했습니다.
  - 갤러리 이미지 선택
  - 이미지 OCR 추출 버튼
  - 바코드 번호 입력 및 미리보기
  - 유효기간 / 카테고리 / 메모 입력
- `barcode_widget` 패키지를 추가해 바코드 미리보기를 지원하도록 정리했습니다.

### 멤버십
- MembershipListScreen을 홈과 같은 디자인 시스템으로 리디자인했습니다.
- MembershipCreateScreen을 이미지 선택 중심의 카드형 입력 화면으로 리디자인했습니다.

### 알림 / 설정
- NotificationListScreen을 최근 알림 리스트 중심 UI로 재정리했습니다.
- NotificationSettingsScreen을 제거하고 SettingsScreen에 알림 설정을 통합했습니다.
- SettingsScreen을 커스텀 헤더 + 베이지 카드형 알림 토글 UI로 리디자인했습니다.

### 기타
- 관련 화면들에 대해 `flutter analyze` 기준 치명적 오류 없이 정리했습니다.
